# ⚠️ Caixa de Alerta

Projeto desenvolvido com HTML e CSS criando uma caixa de alerta interativa com animação de abertura e fechamento.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Praticar a criação de componentes interativos, explorando animações e controle de estado utilizando apenas CSS.

## 📚 O que foi praticado
- Estruturação de componente em HTML  
- Estilização com CSS  
- Criação de caixa de alerta  
- Animação de abertura e fechamento  
- Interação sem uso de JavaScript  
- Uso de checkbox para controle de estado
